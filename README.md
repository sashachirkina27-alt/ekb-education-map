## Данные и источники
OSM: школы, детские сады, районы, граница ГО, УДС, полигоны зданий
Реформа ЖКХ: точки МКД с населением

## Методология
Обеспеченность показывает, сколько мест в школах и детских садах приходится на 1000 жителей в пешеходной доступности от каждого дома. Зоны доступности построены методом изохрон по реальной дорожной сети с радиусом 300 м для детских садов и 500 м для школ в соответствии с СП 42.13330.2016.

## Что было самым сложным
